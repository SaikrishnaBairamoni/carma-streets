Detailed Description

Class to store vehicle information for all vehicles in an intersection. Contains pointer to a status_intent_processor which holds business logic to process status and intent vehicle updates. To implement custom message processing simply extend status_intent_processor and set this object to be the message processor for you vehicle list.
