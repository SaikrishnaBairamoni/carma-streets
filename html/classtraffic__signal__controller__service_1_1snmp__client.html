Parameters

communityThe community id as a string. Defaults to "public" if unassigned.
snmp_versionThe snmp_version as defined in net-snmp.Default to 0 if unassigned. net-snmp version definition: SNMP_VERSION_1:0 SNMP_VERSION_2c:1 SNMP_VERSION_2u:2 SNMP_VERSION_3:3"
timeoutThe time in microseconds after which an snmp session request expires. Defaults to 100 if unassigned

Parameters
input_oidThe OID to request information for.
request_typeThe request type for which the error is being logged. Accepted values are "GET" and "SET" only.
value_intThe integer value for the object returned by reference. For "SET" it is the value to be set. For "GET", it is the value returned for the returned object by reference. This is an optional argument, if not provided, defaults to 0.
value_strString value for the object, returned by reference. Optional argument, if not provided the value is set as an empty string
Returns
Integer value at the oid, returns false if value cannot be set/requested or oid doesn't have an integer value to return.
